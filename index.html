Assets para maquetación

Secretaría de tecnologías para la gestión. Ministerio de Gobierno y Reforma del Estado

Las siguientes pautas y recursos de diseño se aplican a la realización de los trámites, aplicaciones web y servicios interactivos que se publican en el dominio www.santafe.gob.ar, y deben emplearse tanto para la realización de nuevos trámites, aplicativos, como para la normalización de los ya existentes.

Procedimientos

1. Antes de dar inicio a un nuevo desarrollo de trámite en línea se debe comunicar a la Dirección Provincial de Desarrollo de Software de la Secretaría de Tecnologías para la Gestión al siguiente correo dpds@santafe.gob.ar

2. Los recursos y elementos gráficos y visuales (archivos css, js, imágenes, iconos, etc.) que se deben emplear para el desarrollo del trámite se encuentran publicados en este sitio.

3. En la realización del trabajo, los elementos indicados anteriormente deben apuntarse - via url absoluta - al repositorio de assets indicados en este sitio.

4. Una vez finalizado el desarrollo, y antes de su puesta en línea, debe comunicarse a la Dirección Provincial de Desarrollo de Software para la correspondiente validación del cumplimiento de los estándares. En esta instancia se definirá la url de publicación de la aplicación.

5. Para poder ser incluido en el Subportal de Trámites del Sitio Oficial el responsable del desarrollo del trámite en línea deberá contactarse con el contenidista del organismo a fin de que pueda cumplirse con esta instancia y publique el trámite según la pauta establecida. Deberá darle todos los datos del trámite y la url de acceso al formulario online.

La maqueta/plantilla publicada en este sitio muestra a modo de ejemplo cómo aplicar las correspondientes clases de estilos, y como maquetar el contenido html. Se recomienda utilizar Firefox con el complemento Firebug para la inspección de páginas web.
